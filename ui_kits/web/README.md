# COZY-COOL Web UI Kit

Interactive click-through prototype demonstrating the COZY-COOL design system applied to a web interface.

## Screens
- **Home** — Nav, hero carousel, card grid, CTA banner, footer
- **Detail** — Article/detail page with image, body, sidebar
- **Form** — Contact/enquiry form with all input states
- **Components** — Live component library view

## Files
| File | Description |
|---|---|
| `index.html` | Entry point — loads all components and renders the prototype |
| `Nav.jsx` | Top navigation bar with logo, links, search, CTA |
| `Hero.jsx` | Full-bleed hero carousel with overlay text + CTA |
| `Cards.jsx` | Image card grid (3-column) with hover states |
| `Banner.jsx` | Full-width navy CTA banner with orange button |
| `Forms.jsx` | Input/form components with validation states |
| `Footer.jsx` | Footer with links and brand mark |

## Usage
Open `index.html` in a browser. Click nav links and cards to navigate between screens.

## Design Notes
- Primary: `#0D2847` (navy)
- Secondary: `#F2994A` (orange)
- Font: Inter (Google Fonts)
- Icons: Lucide (CDN)
- Grid: 12-column, 30px gutter, max-width 1200px
